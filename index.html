How it works

Unit data is added when swgoh.gg updates the character database. Once a unit is added I update my spreadsheet with the

character location(s), shards dropped for each node, and if available through shipments how many shards are available each day.


To use the calculator simply enter the current star level. This will calculate how many stars are needed to achieve 7 stars.

enter the current amount of shards you have for the star level.

if the character is available through nodes enter how many refreshes you will do for the nodes.

If days remaining shows up as NaN simply select another character and then select desired character for calculation to reresh
